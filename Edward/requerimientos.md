# Mini sistema de ventas

## Descripción
Una herramienta de consola para registrar las ventas diarias de un negocio.

## Qué podría hacer
- registrar una nueva venta con producto, cantidad y total
- listar el historial de ventas
- mostrar el total de ingresos
- buscar ventas por fecha
- eliminar o revertir una venta
- guardar datos en archivo `.json` o `.txt`

## Módulos o archivos sugeridos
- `app.py` o `main.py`
- `funciones.py` o `utils.py`
- carpeta `data/`
- archivo `datos.json` o `datos.txt`
