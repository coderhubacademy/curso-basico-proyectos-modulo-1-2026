# Sistema de inventario básico

## Descripción
Una aplicación que permite llevar el control de los productos de una pequeña tienda o almacén.

## Qué podría hacer
- registrar productos con nombre, cantidad y precio
- listar el inventario actual
- buscar productos por nombre o código
- actualizar la cantidad o precio de un producto
- eliminar productos del inventario
- guardar datos en archivo `.json` o `.txt`

## Módulos o archivos sugeridos
- `app.py` o `main.py`
- `funciones.py` o `utils.py`
- carpeta `data/`
- archivo `datos.json` o `datos.txt`
