# Agenda de contactos

## Descripción
Una aplicación de consola para registrar, gestionar y buscar contactos personales.

## Qué podría hacer
- registrar nombre, teléfono y correo electrónico
- listar todos los contactos
- buscar contactos por nombre
- editar la información de un contacto
- eliminar contactos
- guardar datos en archivo `.json` o `.txt`

## Módulos o archivos sugeridos
- `app.py` o `main.py`
- `funciones.py` o `utils.py`
- carpeta `data/`
- archivo `datos.json` o `datos.txt`

Daniel/
