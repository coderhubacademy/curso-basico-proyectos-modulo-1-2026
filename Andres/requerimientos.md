# Registro de estudiantes y calificaciones

## Descripción
Una aplicación para almacenar notas y calcular promedios de estudiantes en un curso.

## Qué podría hacer
- registrar estudiantes y sus calificaciones
- listar todos los estudiantes con sus notas
- calcular el promedio de notas por estudiante
- editar notas de estudiantes
- eliminar registros de estudiantes
- guardar datos en archivo `.json` o `.txt`

## Módulos o archivos sugeridos
- `app.py` o `main.py`
- `funciones.py` o `utils.py`
- carpeta `data/`
- archivo `datos.json` o `datos.txt`

## Estructura sugerida
```bash
Andres/
├── app.py

├── data/
│   └── datos.json
└── requerimientos.md
```
