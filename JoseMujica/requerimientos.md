# Agenda de eventos personales

## Descripción
Un calendario de consola para llevar el control de compromisos y eventos.

## Qué podría hacer
- registrar eventos con título, fecha y descripción
- listar los próximos eventos
- buscar eventos por fecha o palabra clave
- editar los detalles de un evento
- eliminar eventos pasados o cancelados
- guardar datos en archivo `.json` o `.txt`

## Módulos o archivos sugeridos
- `app.py` o `main.py`
- `funciones.py` o `utils.py`
- carpeta `data/`
- archivo `datos.json` o `datos.txt`
