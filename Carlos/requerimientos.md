# Sistema de biblioteca personal

## Descripción
Una herramienta para gestionar la colección de libros de un usuario.

## Qué podría hacer
- registrar libros con título, autor y género
- listar todos los libros
- buscar libros por título o autor
- marcar libros como leídos o no leídos
- eliminar libros del registro
- guardar datos en archivo `.json` o `.txt`

## Módulos o archivos sugeridos
- `app.py` o `main.py`
- `funciones.py` o `utils.py`
- carpeta `data/`
- archivo `datos.json` o `datos.txt`

## Estructura sugerida
```bash
Carlos/
├── app.py
├── funciones.py
├── data/
│   └── datos.json
└── requerimientos.md
```
