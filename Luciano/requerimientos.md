# Sistema de recetas de cocina

## Descripción
Una aplicación para almacenar, consultar y gestionar tus recetas de cocina favoritas.

## Qué podría hacer
- registrar nuevas recetas con ingredientes y pasos
- listar todas las recetas disponibles
- buscar recetas por nombre o ingrediente
- editar recetas existentes
- eliminar recetas
- guardar datos en archivo `.json` o `.txt`

## Módulos o archivos sugeridos
- `app.py` o `main.py`
- `funciones.py` o `utils.py`
- carpeta `data/`
- archivo `datos.json` o `datos.txt`

## Estructura sugerida
```bash
Luciano/
├── app.py
├── funciones.py
├── data/
│   └── datos.json
└── requerimientos.md
```
