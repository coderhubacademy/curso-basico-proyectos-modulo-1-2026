# Bookit: Una app simple de reservas y citas a TODA HORA (Vzla, Falcón, Pto. Fijo)

## Descripción:

Bienvenido a Bookit, una app de reservas y citas desarrollada por José Petit, puramente a consola en Python


## ¿Qué podría hacer?
- Listar Reservas: Verás la lista de reservas 
- Registrar Reservas: Puedes registrar tu reserva (cliente, fecha, hora) a como quieras y cuando quieras
- Eliminar Reservas: Aquí, puedes eliminar/cancelar tu reserva
- Modificar Reservas: Puedes modificar tu reserva a un calendario más favorable
- Los datos de los clientes están guardados en una base de datos (archivo `.json`)  

## ¿Qué se desea para el usuario?
- Que interactue de forma segura con el programa
- Que pueda reservar o citar en un día y hora favorables a el
- Logre desarrollarse con el programa y ver otras opciones

## ¿Cómo funciona?

- El usuario registra SU reserva, con nombre, fecha y hora 
- Puede modificar o eliminar su reserva (si le conviene)
- Todas esas reservas están listadas en una base de datos (datos.json').
