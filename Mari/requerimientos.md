# Sistema de reservas simple

## Descripción
Una aplicación para gestionar las reservas de citas o servicios por día y hora.

## Qué podría hacer
- registrar nuevas reservas con nombre de cliente, fecha y hora
- listar todas las reservas
- buscar reservas por cliente o fecha
- modificar la fecha u hora de una reserva
- cancelar (eliminar) reservas
- guardar datos en archivo `.json` o `.txt`

## Módulos o archivos sugeridos
- `app.py` o `main.py`
- `funciones.py` o `utils.py`
- carpeta `data/`
- archivo `datos.json` o `datos.txt`

Mari/
